My Todo List

My to-do list is an Android software developed based on Kotlin. The purpose of this app is to allow users to record things that to do later. Let users avoid forgetting something important.

The user can use the add button to add things that need to do later.

If the user completes something in the todo list, user can click the check mark to make the completed thing invisible. Similarly, user can also delete the things that cannot be done.

On the analytics page, users can see all completed and deleted content since using the app

Users can also view the percentage of the things that need to be done, the things that have been done and the things that cannot be done.
