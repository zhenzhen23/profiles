Dream Chaser

The main idea of this app is to help users get rid of procrastination in the form of daily tasks.

There are five main sections, home page, event page, pomodoro clock page, todo list page, and settings interface.

In the home page, the title shows the end goal and countdown to estimated completion date set by the user.

The upper part of the main page is the event section, which includes all the events set by the user. The lower part is the goal section, which shows the percentage of goal completion. When the user completes the daily event, the user can check the checkbox corresponding to the event, The progress bar of the corresponding goal will also increase.

On the event page, users can set the event and goal according to their needs.

When the user needs to concentrate on the work, the user can use the Pomodoro Clock in the app. And user will be locked in the Pomodoro Clock page until the user resets the clock. User can also pause the clock when they need.

When the user concentrates on work for 25 minutes, the app will remind the user to take 5 minutes break. After 4 cycles, the user will get a slightly longer break.

Users can also add events that need to be completed on the todolist page to avoid forgetting

Finally, on the settings page the user can change the information or reset the entire software
